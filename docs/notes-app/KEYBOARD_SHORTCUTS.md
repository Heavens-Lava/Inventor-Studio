## Tips

### Markdown Shortcuts
Type these patterns for quick formatting:
- `# ` → Heading 1
- `## ` → Heading 2
- `### ` → Heading 3
- `- ` or `* ` → Bulleted list
- `1. ` → Numbered list
- `- [ ] ` → Task list
- `` ` ` `` → Inline code
- ` ``` ` → Code block
- `> ` → Blockquote
- `---` → Horizontal rule

### Voice Commands
When voice input is active, say:
- "New paragraph" → Start new paragraph
- "Period" / "Comma" / "Question mark" → Punctuation
- "New line" → Line break
- "Bold that" → Bold last word/phrase
- "Italic that" → Italicize last word/phrase
- "Undo that" → Remove last input

### Multi-Selection
- `Ctrl/Cmd + Click` → Add to selection
- `Shift + Click` → Select range
- `Ctrl/Cmd + A` → Select all

## Platform Differences

### Mac (⌘ Command)
Replace `Ctrl` with `⌘ Command (Cmd)` in all shortcuts above.

### Windows/Linux (Ctrl)
Use `Ctrl` as shown in the shortcuts above.

### Alternative Keys
- `Alt` on Windows/Linux = `Option (⌥)` on Mac
- `Ctrl` on Windows/Linux = `Command (⌘)` on Mac
